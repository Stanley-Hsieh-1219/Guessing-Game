# 🎯 猜謎大挑戰

一款互動式猜謎遊戲，透過提示猜出正確答案！

🔗 **[線上遊玩](https://stanley-hsieh-1219.github.io/Guessing-Game/)**

---

## 🎮 遊戲特色

- **5 種挑戰類別**：🍽️ 美食 ｜ 🚗 交通工具 ｜ 🦁 陸地動物 ｜ 🐳 海洋動物 ｜ 🏙️ 世界知名城市
- **3 種難度等級**：🥄 新手 ｜ 🍳 資深 ｜ ⭐ 大師
- **300 題題庫**：每個類別 × 難度各 20 題
- **每回合 10 題**：從題庫隨機抽選
- **2 次作答機會**：首次答對 +10 分，第二次 +5 分

## 🆕 輔助功能按鈕

遊玩過程中可使用三個輔助按鈕：

| 按鈕           | 功能                             | 限制            |
| -------------- | -------------------------------- | --------------- |
| 💡 再一個提示   | 顯示該謎底的額外相關提示         | 每回合限用 1 次 |
| 🔤 給我第一個字 | 揭露答案的第一個字               | 每回合限用 1 次 |
| 🏳️ 放棄         | 放棄該題（0 分），直接進入下一題 | 無次數限制      |

> 使用過的按鈕會自動反灰，直到下一回合才能再次使用。

## 📖 遊戲流程

1. 點擊「開始遊戲」
2. 選擇挑戰類別（美食 / 交通工具 / 陸地動物 / 海洋動物 / 世界知名城市）
3. 選擇難度等級（新手 / 資深 / 大師）
4. 根據三條提示輸入答案，可使用輔助按鈕獲得更多線索
5. 完成 10 題挑戰，查看最終分數與答題紀錄

## 🛠️ 技術

- 單一 `index.html` 檔案，無需後端
- Vanilla JavaScript + Tailwind CSS CDN
- 純前端狀態機架構（Start → Category → Difficulty → Play → Result）
