# 🎯 猜謎大挑戰

一款互動式猜謎遊戲，透過三條提示猜出正確答案！

🔗 **[線上遊玩](https://stanley-hsieh-1219.github.io/Guessing-Game/)**

---

## 🎮 遊戲特色

- **5 種挑戰類別**：🍽️ 美食 ｜ 🚗 交通工具 ｜ 🦁 陸地動物 ｜ 🐳 海洋動物 ｜ 🏙️ 世界知名城市
- **3 種難度等級**：🥄 新手 ｜ 🍳 資深 ｜ ⭐ 大師
- **300 題題庫**：每個類別 × 難度各 20 題
- **每回合 10 題**：從題庫隨機抽選
- **2 次作答機會**：首次答對 +10 分，第二次 +5 分

## 📖 遊戲流程

1. 點擊「開始遊戲」
2. 選擇挑戰類別（美食 / 交通工具 / 陸地動物 / 海洋動物 / 世界知名城市）
3. 選擇難度等級（新手 / 資深 / 大師）
4. 根據三條提示輸入答案，完成 10 題挑戰
5. 查看最終分數與答題紀錄

## 🛠️ 技術

- 單一 `index.html` 檔案，無需後端
- Vanilla JavaScript + Tailwind CSS CDN
- 純前端狀態機架構（Start → Category → Difficulty → Play → Result）
